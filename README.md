# Next.js Boilerplate

Next.js 기반 프론트엔드 프로젝트를 빠르게 시작하기 위한 Boilerplate입니다.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- React Query
- Zustand
- ESLint
- Prettier
- Husky
- lint-staged

## Folder Structure

src
├ app
├ features
├ shared
├ assets
└ styles

## Getting Started

install

npm install

run dev server

npm run dev

## Scripts

npm run dev
npm run build
npm run start
npm run lint
npm run format
npm run type-check
